# Бор (Trie)

## Что такое бор

**Бор** — дерево, в котором каждое ребро помечено символом, а каждый путь от корня до узла задаёт некоторый префикс множества строк. Листья (или помеченные узлы) соответствуют конечным строкам.

```
Строки: {"abc", "ab", "abd", "b", "bc"}

         root
        /    \
       a      b *
       |     / 
       b *  c *
      / \
     c * d *
```

`*` — узел помечен как конец строки.

---

## Структура узла

```cpp
struct TrieNode {
    int children[26];   // индексы дочерних узлов (-1 = нет)
    bool is_end;        // конец какой-либо строки

    TrieNode() : is_end(false) {
        fill(children, children + 26, -1);
    }
};
```

Весь бор — вектор узлов:

```cpp
struct Trie {
    vector<TrieNode> nodes;

    Trie() { nodes.emplace_back(); }  // корень — nodes[0]

    int root() const { return 0; }
};
```

---

## Вставка строки — $O(|s|)$

```cpp
void insert(const string& s) {
    int cur = 0;
    for (char ch : s) {
        int c = ch - 'a';
        if (nodes[cur].children[c] == -1) {
            nodes[cur].children[c] = nodes.size();
            nodes.emplace_back();
        }
        cur = nodes[cur].children[c];
    }
    nodes[cur].is_end = true;
}
```

Идём по символам строки. Если ребра нет — создаём новый узел. В конце помечаем узел.

---

## Поиск строки — $O(|s|)$

```cpp
bool search(const string& s) {
    int cur = 0;
    for (char ch : s) {
        int c = ch - 'a';
        if (nodes[cur].children[c] == -1) return false;
        cur = nodes[cur].children[c];
    }
    return nodes[cur].is_end;
}
```

### Поиск префикса (есть ли хоть одно слово с таким началом):

```cpp
bool starts_with(const string& prefix) {
    int cur = 0;
    for (char ch : prefix) {
        int c = ch - 'a';
        if (nodes[cur].children[c] == -1) return false;
        cur = nodes[cur].children[c];
    }
    return true;  // не проверяем is_end — нас устраивает любой путь
}
```

---

## Удаление строки — $O(|s|)$

Простой вариант — просто снять флаг `is_end`. Узлы не удаляем (они могут быть нужны другим строкам).

```cpp
bool remove(const string& s) {
    int cur = 0;
    for (char ch : s) {
        int c = ch - 'a';
        if (nodes[cur].children[c] == -1) return false; // строки нет
        cur = nodes[cur].children[c];
    }
    if (!nodes[cur].is_end) return false;
    nodes[cur].is_end = false;
    return true;
}
```

Если нужно удалять и узлы — обходим рекурсивно и чистим «висячие» вершины (без детей и без флага).

---

## Сложности

| Операция | Время | Память |
|----------|-------|--------|
| Вставка | $O(\|s\|)$ | — |
| Поиск | $O(\|s\|)$ | — |
| Удаление (флаг) | $O(\|s\|)$ | — |
| Память всего бора | — | $O(\text{суммарная длина строк} \cdot \|\Sigma\|)$ |

Главный минус — **память**. Каждый узел хранит массив из $|\Sigma|$ указателей, даже если большинство пустые. Для большого алфавита используют `unordered_map<char, int>` вместо массива.

---

## Альтернативное хранение (экономичное)

```cpp
struct TrieNode {
    unordered_map<char, int> children;
    bool is_end = false;
};
```

Медленнее по константе, но память — $O(\text{суммарная длина строк})$.

## Применения

### 1. Автодополнение / словарь

Вставляем все слова в бор. Для запроса-префикса `q`:
1. Доходим до узла `q` за $O(|q|)$.
2. Обходим поддерево (BFS/DFS) и собираем все слова.

### 2. Лексикографически наименьшая строка, не входящая в множество

Обходим бор жадно — на каждом шаге берём наименьший доступный символ.

### 3. Хранение IP-адресов (двоичный бор)

Бор по битам числа. Эффективный поиск по маске.

### 4. Максимальный XOR двух чисел (двоичный бор)

Вставляем числа побитно (от старшего). Для числа $x$ жадно выбираем противоположный бит — максимизируем XOR.

```cpp
// Двоичный бор: биты от 29 до 0
void insert_num(int x) {
    int cur = 0;
    for (int bit = 29; bit >= 0; bit--) {
        int b = (x >> bit) & 1;
        if (nodes[cur].children[b] == -1) {
            nodes[cur].children[b] = nodes.size();
            nodes.emplace_back();
        }
        cur = nodes[cur].children[b];
    }
}

int max_xor(int x) {
    int cur = 0, res = 0;
    for (int bit = 29; bit >= 0; bit--) {
        int b = (x >> bit) & 1;
        int want = 1 - b;   // хотим противоположный бит
        if (nodes[cur].children[want] != -1) {
            res |= (1 << bit);
            cur = nodes[cur].children[want];
        } else {
            cur = nodes[cur].children[b];
        }
    }
    return res;
}
```

### 5. Основа алгоритма Ахо-Корасик

Бор + суффиксные ссылки = поиск множества паттернов в тексте за $O(n + \text{сумма длин паттернов})$.

## Полная реализация

```cpp
struct Trie {
    struct Node {
        int ch[26];
        bool end = false;
        Node() { fill(ch, ch + 26, -1); }
    };
    vector<Node> t = {Node()};

    void insert(const string& s) {
        int v = 0;
        for (char c : s) {
            int i = c - 'a';
            if (t[v].ch[i] < 0) { t[v].ch[i] = t.size(); t.emplace_back(); }
            v = t[v].ch[i];
        }
        t[v].end = true;
    }

    bool find(const string& s) {
        int v = 0;
        for (char c : s) {
            int i = c - 'a';
            if (t[v].ch[i] < 0) return false;
            v = t[v].ch[i];
        }
        return t[v].end;
    }
};
```

## Типичные вопросы

**В.** В чём отличие бора от хеш-таблицы строк?  
**О.** Бор поддерживает **префиксные запросы** (все слова с префиксом $q$) и лексикографический обход — хеш-таблица нет.

**В.** Сколько узлов в боре?  
**О.** Не более суммарной длины всех вставленных строк плюс 1 (корень).

**В.** Что будет, если вставить одну строку дважды?  
**О.** Путь уже существует, только `is_end` установится повторно — изменений не будет.

**В.** Как обойти все строки бора в лексикографическом порядке?  
**О.** DFS от корня, перебирая детей в порядке `0, 1, ..., 25` — символы идут по алфавиту.
