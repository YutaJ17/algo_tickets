## 15. Диаметр дерева

### Теория

**Дерево** - связный ацикличный граф.
**Диаметр дерева** - максимальный эксцентриситет.
**Эксцентриситет вершины v** - расстояние до наиболее удаленной от `v` вершины.
**Расстояние** - длина кратчайшей цепи.
**Длина пути** - число ребер в нем.

### Принцип работы

1. Выбираем любую вершину `S`
2. Проходим DFS/BFS, стартуя от `S`, находим наиболее удаленную вершину от стартовой - `A`.
3. Повторяя пункт 2, находим наиболее удаленную вершину от `A` - `B`.
4. Количество ребер в цепи (A, B) и будет исходным диаметром.

### Структуры данных

- Список смежности для взвешенных (`vector<vector<pair<v, w>>>`) и невзвешенных (`vector<vector<v>>`) графов.
- Для восстановления пути - массив предков из DFS/BFS.


### Уточнения

- Если диаметр четный - один центр дерева, если нечетный - два.

### Реализация на C++

```cpp
#include <iostream>
#include <vector>
#include <stack>

class Graph {
    int V;
    std::vector<std::vector<int>> adj;

public:
    Graph(int V): V(V) {
        adj.resize(V);
    }

    void addEdge(int v1, int v2) {
        adj[v1 - 1].push_back(v2 - 1);
        adj[v2 - 1].push_back(v1 - 1);
    }

    std::pair<int, int> find_diam(int start, std::vector<int>& parent) {  // parent для восстановления пути
        std::vector<int> dist(V, -1);
        parent.assign(V, -1);
        std::stack<int> st;
        dist[start] = 0;
        st.push(start);
        while (!st.empty()) {
            int curr = st.top();
            st.pop();
            for (int neighbor : adj[curr]) {
                if (dist[neighbor] == -1) {
                    dist[neighbor] = dist[curr] + 1;
                    parent[neighbor] = curr;  // заполняем массив предков
                    st.push(neighbor);
                }
            }
        }
        int v_end;  // конец простой цепи
        int d = 0;     // диаметр
        for (int i = 0; i < V; ++i) {
            if (d < dist[i]) {
                d = dist[i];
                v_end = i;
            }
        }
        return {d, v_end};
    }
};

int main() {
    Graph g(7);
    g.addEdge(1, 2);
    g.addEdge(1, 3);
    g.addEdge(1, 4);
    g.addEdge(4, 5);
    g.addEdge(5, 6);
    g.addEdge(5, 7);

    std::vector<int> parent1;
    auto dfs1 = g.find_diam(0, parent1);

    std::vector<int> parent2;
    auto dfs2 = g.find_diam(dfs1.second, parent2);

    int diam = dfs2.first;
    std::cout << diam << '\n';  // диаметр

    int st = dfs1.second;
    int end = dfs2.second;

    std::vector<int> path;  // восстанавливаем путь
    int curr = end;
    while (curr != -1) {
        path.push_back(curr);
        if (curr == st) break;
        curr = parent2[curr];
    }

    for (int i : path) {
        std::cout << i + 1 << " ";
    }

}
```
